Para testar o nosso Counting Bloom Filter, usaremos o mesmo raciciocinio seguido nos guioes praticos: inserimos n strings aleatorias no filtro e, posteriormente, procuramos m strings aleatorias nesse mesmo filtro.
A probabilidade de serem geradas 2 strings iguais e desprezivel, logo o acontecimento de uma string gerada ser de facto membro do cbf e reduzido ao impossivel.
Sempre que for detetada uma string no cbf consideramos como falso positivo!
