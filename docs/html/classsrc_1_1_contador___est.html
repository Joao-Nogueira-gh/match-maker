Detailed Description

Classe para desenvolvimento do módulo do 'Contador Estocástico', um contador que só é incrementado com base numa probabilidade.

src.Contador_Est.Contador_Est ( double  prob)

Um construtor unico que executa o trabalho de dois construtores diferentes.
Caso o valor passado seja menor que 1, estamos a trabalhar com uma probabilidade, caso seja maior que 1 estamos a trabalhar com um fator.
